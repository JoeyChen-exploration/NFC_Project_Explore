# 极致克制美学任务文档

## 🎯 设计哲学

**少即是多，空即是满，静即是动**

## 🖤 核心原则

### 1. 文字极简主义

- 能用图标+2词表达，绝不用句子
- 删除所有解释性文字
- 标题最大5词，描述最大8词
- 无段落，只有短语

### 2. 留白即设计

- 元素间距 ≥ 64px
- 卡片内边距 ≥ 32px
- 页面边缘留白 ≥ 80px
- 呼吸感 > 信息密度

### 3. 黑白主调

- 主色: 黑白灰 (90% 以上)
- 强调色: 极少量 (≤ 5% 面积)
- 无渐变，无鲜艳色彩
- 质感来自光影，而非颜色

### 4. 网格系统

- 12列网格系统
- 严格对齐
- 比例协调 (黄金分割)
- 视觉平衡

### 5. 精致细节

- 1px 精致边框
- 微妙的阴影层次
- 完美的圆角比例
- 流畅的缓动动画

## 🛠️ 技术实施

### 1. 删除所有多余装饰

- 移除渐变背景
- 移除玻璃态效果
- 移除复杂动画
- 移除彩色元素

### 2. 建立严格的网格系统

```css
.grid-12 {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: var(--space-8);
}

.col-4 {
  grid-column: span 4;
}
.col-6 {
  grid-column: span 6;
}
.col-8 {
  grid-column: span 8;
}
.col-12 {
  grid-column: span 12;
}
```

### 3. 创建极致组件库

```jsx
// 极致按钮

// 极致输入框

// 极致卡片

  <Icon size={64} />
  <h3>Links</h3>
</MinimalCard>
```
